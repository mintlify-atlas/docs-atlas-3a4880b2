## GET /system

Returns comprehensive environment information useful for debugging, auditing, and support. Covers the WordPress installation, PHP runtime, database connection, web server, currently active plugins, key WordPress constants, and the next 20 scheduled cron events.

```bash
curl -X GET "https://yoursite.com/wp-json/wp-manager-pro/v1/system" \
  -H "X-WP-Nonce: YOUR_NONCE"
```

### Response fields

<ResponseField name="wordpress" type="object" required>
  WordPress installation details.
  <Expandable title="properties">
    <ResponseField name="version" type="string">WordPress version.</ResponseField>
    <ResponseField name="site_url" type="string">Site URL (`get_site_url()`).</ResponseField>
    <ResponseField name="home_url" type="string">Home URL (`get_home_url()`).</ResponseField>
    <ResponseField name="admin_email" type="string">Admin email address.</ResponseField>
    <ResponseField name="language" type="string">Site locale, e.g. `"en_US"`.</ResponseField>
    <ResponseField name="timezone" type="string">WordPress timezone string, e.g. `"America/New_York"`.</ResponseField>
    <ResponseField name="charset" type="string">Blog character set, e.g. `"UTF-8"`.</ResponseField>
    <ResponseField name="multisite" type="boolean">`true` if WordPress Multisite is enabled.</ResponseField>
    <ResponseField name="memory_limit" type="string">`WP_MEMORY_LIMIT` constant value.</ResponseField>
    <ResponseField name="max_memory" type="string">`WP_MAX_MEMORY_LIMIT` constant value, or `"N/A"` if not defined.</ResponseField>
    <ResponseField name="debug" type="boolean">`true` if `WP_DEBUG` is enabled.</ResponseField>
    <ResponseField name="debug_log" type="boolean">`true` if `WP_DEBUG_LOG` is enabled.</ResponseField>
    <ResponseField name="debug_display" type="boolean">`true` if `WP_DEBUG_DISPLAY` is enabled.</ResponseField>
    <ResponseField name="table_prefix" type="string">Database table prefix, e.g. `"wp_"`.</ResponseField>
    <ResponseField name="uploads_dir" type="string">Absolute path to the uploads base directory.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="cron" type="object[]" required>

  <Expandable title="item properties">
    <ResponseField name="hook" type="string">WordPress action hook name for the cron event.</ResponseField>

    <ResponseField name="timestamp" type="number">Unix timestamp of the next scheduled run.</ResponseField>
  </Expandable>
</ResponseField>
