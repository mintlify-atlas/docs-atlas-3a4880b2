## What is WP Manager Pro?

**WP Manager Pro** replaces the need for multiple separate admin plugins by providing a single, fast, modern interface for managing every critical aspect of a WordPress site. Install it as a standard WordPress plugin and a full React-powered control panel appears under your WP Admin menu.

All operations run through a secured REST API namespace (`wp-manager-pro/v1`) that requires the `manage_options` capability — so only site administrators can access it.

## Requirements

| Requirement | Minimum |
|-------------|---------|
| WordPress   | 5.9+    |
| PHP         | 7.4+    |
| MySQL/MariaDB | 5.6+  |
| Browser     | Modern (ES2020+) |

<Note>
  WP Manager Pro requires the `manage_options` WordPress capability. Only administrators can access the dashboard and all REST API endpoints enforce this requirement.
</Note>

## Built with modern technology

WP Manager Pro is built with **React 19**, **TypeScript**, **Vite**, **Tailwind CSS**, and **shadcn/ui** on the frontend, backed by a PHP REST API running on the WordPress REST API infrastructure. The Monaco Editor (VS Code engine) powers the File Manager and Code Snippets editor for a native development experience inside your WordPress admin.
