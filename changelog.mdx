  <Accordion title="v2.9.2 — Media picker fix & logo select button" defaultOpen>
    **Released:** 2026-03-17

    #### Fixed

    - **Media Library picker** — all "Select image" / "Media" buttons now reliably open the native WordPress Media Library modal instead of falling back to a browser `prompt()`. Root cause: `window.wp.media` was not yet callable at the time the React bundle first evaluated the check. Fix: a bridge function (`window.wmpOpenMedia`) is now injected via `wp_add_inline_script('after')`, which runs after all media scripts are fully initialised; React calls this bridge instead of calling `wp.media` directly.
    - **Settings → Branding → Custom Logo URL** — added a dedicated **Select** button that opens the WordPress Media Library to pick the sidebar logo image, consistent with all other media pickers in the plugin.
  </Accordion>

  <Accordion title="v2.9.1 — Login page & Coming Soon redesign">
    **Released:** 2026-03-17

    #### Fixed

    - **Login Page CSS not applying** — switched from inline `<style>` inside `login_enqueue_scripts` to `wp_add_inline_style('login', $css)` so custom styles are injected after WordPress's own login stylesheet.
    - **Media button falling back to `prompt()`** — `wp_enqueue_media()` is now called in `Admin::enqueue_assets()`, making `window.wp.media` available in the React app for logo and background image pickers.

    #### Added

    - **Login Page — Privacy & Legal Links** — toggle to show the site's built-in Privacy Policy link below the login form; custom links HTML textarea for Terms of Service, Cookie Policy, or any anchor tags (passed through `wp_kses_post`).
    - **Coming Soon — UI redesign** — side-by-side settings and live preview layout matching the Login Page; Media Library picker for logo and background image; content, colour, and email-capture sections reorganised into separate cards.
    - **Coming Soon — rendered page redesign** — modern countdown with individual Day / Hours / Min / Sec glass-card tiles; background image support; fluid typography with `clamp()` font sizes.
  </Accordion>

  <Accordion title="v2.9.0 — Developer Utilities">
    **Released:** 2026-03-17

    #### Added

    - **Developer Utilities page** (`/developer`) — new sidebar page with a 6-tab layout for advanced WordPress introspection and testing.
    - **Hook Explorer** — browses all registered `add_action`/`add_filter` hooks from `$wp_filter`; shows hook name, priority, callback name with source file and line number via Reflection; searchable.
    - **REST API Tester** — built-in Postman-style client pre-authenticated with WordPress nonce; browse registered routes by namespace; set method, path, JSON body, and headers; response panel shows status, duration, headers, and formatted body.
    - **Dummy Data Generator** — create up to 50 test posts, pages, users, or WooCommerce products; items tagged with `_wmp_dummy_data` meta for safe bulk one-click cleanup.
    - **Rewrite Rules Tester** — input any URL path and see the matched rule, redirect query string, and resolved `$query_vars`; expandable full rules table.
    - **Object Cache Browser** — lists Redis keys via SCAN or WP internal cache entries by prefix; inspect value per key; delete individual keys.
    - **Database Prefix Changer** — safely rename the `wp_` prefix with dry-run preview, `RENAME TABLE` for all tables, updates `option_name` and `meta_key` rows, rewrites `$table_prefix` in `wp-config.php`; confirmation requires checkbox and typed new prefix.
    - **Agency Tools → Login Page** redesigned with side-by-side settings and persistent live preview; Media Library picker for logo and background image; colour swatches with native picker overlay.
  </Accordion>

  <Accordion title="v2.7.1 — Self-update system">
    **Released:** 2026-03-16

    #### Added

    - **Self-Update System** — integrates WP Manager Pro with WordPress's native plugin update mechanism via GitHub Releases API (12-hour cache); shows "View Release Notes" link in the Plugins list update row.

    #### Fixed

    - Update Manager showing false "Done" for premium/unlicensed plugins — `null` result from `Plugin_Upgrader::upgrade()` now correctly identified as no-update; added distinct badges: "License Required" (amber) and "Update Unavailable" (grey).
    - 2FA QR code not rendering — switched from deprecated Google Charts QR API to `api.qrserver.com`.
    - Custom login logout not working — `logout` action added to the bypass array; `logout_redirect` filter added to redirect to the custom login URL after logout.
    - Plain Permalinks warning banner added across all plugin pages (REST API requires pretty permalinks).
  </Accordion>

  <Accordion title="v2.7.0 — Security Scanner">
    **Released:** 2026-03-16

    #### Added

    - **Security Scanner page** (`/security-scanner`) — dedicated 4-tab page with a unified security score.
    - **Overview tab** — animated security score ring (0–100, letter grade A+ to F); four summary cards; "Run All Scans" button.
    - **Malware Scanner** — scans up to 8,000 PHP/JS/HTML files against 13 detection patterns including `eval(base64_decode(...))`, `preg_replace` with `/e` modifier, known webshell markers (FilesMan, r57, c99), and `document.write(unescape(...))` JS injection; files >512 KB skipped.
    - **Vulnerability Database** — checks plugins and themes against the WPScan CVE API; shows CVE title, CVSS score, fix version, and reference links; configurable API key.
    - **SSL Monitor** — reads certificate via `stream_socket_client` + `openssl_x509_parse`; shows subject, issuer, SAN entries, validity dates, and days remaining with colour-coded alerts.
    - **Core & PHP tab** — checks WordPress version against latest; PHP EOL status for versions 5.6 through 8.4.
  </Accordion>

  <Accordion title="v2.6.0 — Update Manager">

  <Accordion title="v2.0.0 — Security Suite">
    **Released:** 2026-03-10

    #### Added

    - Security page fully redesigned with 5-tab layout: Overview, Login, Hardening, Integrity, Two-Factor Auth.
    - **Login Attempt Limiter** — configurable max attempts, counting window, and lockout duration; lockout log with per-IP unlock.
    - **Disable XML-RPC** toggle.
    - **Hide WordPress Version** toggle — removes version from `<meta name="generator">` and strips `?ver=` from asset URLs.
    - **IP Blocklist** — exact IPs and CIDR ranges; enforced on `init` hook.

    - **TOTP 2FA** — RFC 6238, SHA1, 6-digit codes, 30-second window; 8 one-time backup codes stored as MD5 hashes; 100% native PHP.
